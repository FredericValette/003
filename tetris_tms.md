**Prompt**
ich möchte gern in ein  spa html seite ein tetris ähnlich spiele mit als hinergrund ein abläufe diagram und anstatt wurfel wurde ich gern bugs die sich als fixed explodieren wenn sie alle auf ein linie sind.

**Hintergrund (Ablauf-Diagramm)**
- Animiertes SVG-Flussdiagramm mit dem kompletten Software-Lifecycle: Requirements → Design → Coding → Code Review → Testing → Deploy → Monitoring
- Bug-Tracking-Flow: 🐛 Bug Found → Bug Report → Analysis → 🔧 Bug Fix → ✅ Verified
- Kanban-Board, Code-Snippets und Log-Meldungen als dekorative Elemente
- Animierte gepunktete Pfeile mit CSS-Animationen

**Spiel-Mechanik**
- 7 Spielsteine als Bugs: 🐛 🐜 🐞 🦗 🦟 🦂 🪲 (je ein anderes Emoji)
- Wenn eine Zeile voll ist → **Explosion**: Partikel mit 💥 ✅ ⚡ 🔥 fliegen aus
- "✅ BUG FIXED!" / "💥 DOUBLE FIX!" / "🎉 TETRIS!" schwebt auf
- Ghost-Piece zeigt wo der Bug landet
- Nächster Bug als Vorschau

**Steuerung**
- ← → Bewegen, ↑ Drehen, ↓ Soft Drop, Leertaste Hard Drop, P Pause
- Touch-Wischen und On-Screen-Buttons für Mobil
